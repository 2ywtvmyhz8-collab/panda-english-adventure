# Panda English Adventure V0.3

新增：
- 防亂猜機制：答錯選項會鎖住。
- 第二次錯後出現 Hint。
- 三星評分：第一次答對 3 星；第二次 2 星；第三次後 1 星。
- 錯題箱 Review Box：錯過的題目會被記錄。
- 媽媽後台新增錯題與星星表現。

更新方式：
1. 解壓縮 zip。
2. 到 GitHub repository：panda-english-adventure。
3. Add file → Upload files。
4. 上傳 index.html、style.css、game.js、README.md。
5. Commit changes。
6. 等 1–2 分鐘刷新網站。
