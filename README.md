# Fox Academy Modular v1

這版是「架構整理版」。

## 檔案結構
- index.html：頁面骨架，之後盡量不改。
- css/style.css：畫面樣式。
- data/unit3.js：Unit 3 題庫。
- data/preview.js：預習資料。
- js/store.js：存檔。
- js/ui.js：畫面、發音、洗牌。
- js/quiz.js：題型引擎。
- js/app.js：學習流程。

## 以後怎麼更新
- 新增 Unit 4：新增 `data/unit4.js`，再讓 index 載入。
- 新增題目：只改 `data/unit3.js` 或未來的 `unit4.js`。
- 新增題型：只改 `js/quiz.js`。
- 新增畫面：才改 `index.html`。

上傳 GitHub 時，要整包上傳這些資料夾與檔案。
